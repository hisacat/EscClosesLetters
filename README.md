# <img src="./About/ModIcon.png" style="height: 1em;" /> Esc Closes Letters

![Preview](./About/Preview.png)  
[English](./README.md) | [한국어](./README.ko.md)

A RimWorld mod that lets you close letter popups with the `Cancel` and `Accept` keys

## Features

- Dismiss letter popups with the `Cancel` key or the `Accept` key. You can toggle each one independently in the mod settings.
- Eligible dismiss options are vanilla-style choices such as `OK`, `Close`, `Postpone`, and `Reject` (labels match your game language). Choose which types apply in the mod settings.
  - `OK` and `Close` are enabled by default; the others are off by default.

## Supported RimWorld versions

- `1.4`, `1.5`, `1.6`

## Requirements

- [Harmony](https://steamcommunity.com/sharedfiles/filedetails/?id=2009463077)

## Compatibility

Should be compatible with most mods.

## Save compatibility

This mod does not modify save files, so you can add or remove it at any time.
